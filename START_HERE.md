## 🚀 Deploy in 5 Minutes

### Option 1: Vercel (Easiest)
1. Push to GitHub
2. Go to vercel.com
3. Import your repo
4. Click Deploy
5. Done! ✨

### Option 2: Netlify
1. Run `npm run build`
2. Drag `dist` folder to netlify.com
3. Done! ✨

**Full deployment guide in DEPLOYMENT.md**
