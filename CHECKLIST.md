## 📝 Notes

### When Adding New Projects

1. Take a screenshot of your project
2. Add image to `/public` folder
3. Update `src/data/portfolio.js`:
   ```js
   {
     id: 4,
     title: "Project Name",
     description: "What it does",
     tech: ["React", "Tailwind"],
     liveUrl: "https://...",
     githubUrl: "https://...",
     image: "/project4.jpg",
     featured: false,
   }
   ```

### Getting Feedback

Ask reviewers to check:
- First impression
- Navigation clarity
- Content readability
- Mobile experience
- Loading speed
- Any bugs
