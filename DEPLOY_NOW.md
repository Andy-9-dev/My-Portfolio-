## 🎯 3 Simple Steps

### Step 1: Install Dependencies & Test Build
```bash
npm install
npm run build
```

If build succeeds, you're ready! ✅

### Step 2: Push to GitHub
```bash
# Initialize git
git init

# Add all files
git add .

# Commit
git commit -m "Portfolio ready for deployment"

# Create repo on GitHub, then:
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git branch -M main
git push -u origin main
```

### Step 3: Deploy on Vercel
1. Go to [vercel.com](https://vercel.com)
2. Sign in with GitHub
3. Click "Add New" → "Project"
4. Select your repository
5. Click "Deploy"

**Done! Your site will be live in ~2 minutes!** 🎉

## 🔗 After Deployment

1. **Get Your URL**
   - Vercel gives you: `your-project.vercel.app`

2. **Update URLs** (Optional)
   - Replace `your-domain.vercel.app` in `index.html` with your actual URL
   - Update `public/robots.txt` with your URL

3. **Share Your Portfolio**
   - Add to LinkedIn
