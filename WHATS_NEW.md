## 🎯 How to Use

### Installation
```bash
npm install
npm run dev
```

### Navigation
- Click navbar links to navigate
- Each page loads with unique animation
- Smooth transitions between pages
- Mobile menu closes automatically

### Customization
- Edit page content in `src/pages/`
- Modify animations in each page file
- Update personal data in `src/data/portfolio.js`
